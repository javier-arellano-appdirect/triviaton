# Sound Files

This directory should contain the following MP3 sound files for the game:

## Required Sound Files

1. **category-reveal.mp3** - Played when each category is revealed during the board animation
2. **daily-double.mp3** - Played when a Daily Double question is revealed
3. **game-start.mp3** (optional) - Can be played when the game starts
4. **correct-answer.mp3** (optional) - Can be played when a correct answer is given
5. **wrong-answer.mp3** (optional) - Can be played when a wrong answer is given
6. **game-over.mp3** (optional) - Can be played when the game ends

## Current Status

**PLACEHOLDER FILES NEEDED**: Please add your MP3 files to this directory.

The game will attempt to play these sounds at appropriate times. If the files are not present, the game will continue to function normally without sound.

## How to Add Sounds

1. Obtain or create MP3 files for the sounds listed above
2. Place them in this directory (`public/sounds/`)
3. Ensure the filenames match exactly (case-sensitive)
